# 三Agent思想辩论场 — 用户使用指南

## 快速开始

在飞书中输入以下命令：

```
/debate 自由是什么
```

三位思想家将进行3轮辩论。

## 命令列表

| 命令 | 说明 | 示例 |
|------|------|------|
| `/debate [话题]` | 发起3轮辩论 | `/debate 什么是幸福` |
| `/debate_rounds [3\|6\|9] [话题]` | 指定轮次 | `/debate_rounds 6 科技让人类更幸福吗` |
| `/debate_status` | 查看最近辩论进度 | |
| `/debate_history` | 过往辩论记录 | |

## 角色介绍

| 角色 | 派别 | 一句话 |
|------|------|--------|
| 🌀 庄周 | 道家 | "连'自己'也是幻觉——何必执着？" |
| ⚡ 尼采 | 权力意志 | "成为你自己！超越人！" |
| 🔍 波伏娃 | 存在主义 | "你首先需要一个可以成为自己的处境" |

## 辩论流程

1. **第1轮** — 三人各自发表立场声明
2. **第2轮** — 回应+深化。本轮最精彩：庄子vs尼采的消解vs超越碰撞
3. **第3轮** — 收束：最后一击或开放问题

## 最佳碰撞示例

### 例1: 自由是什么（自由辩论记录片段）

**庄周**: 自由？你们在问一条鱼，水是不是牢笼。我梦见蝴蝶，不知是庄周梦蝶还是蝶梦庄周——连'我是谁'尚且未定，何谈自由？

**尼采**: 哈！你这条滑溜溜的东方泥鳅！你说自由是笼子？不对！大鹏之所以飞九万里，是因为它必须飞——那是它权力意志的自我表达！你的逍遥不过是逃避！

**波伏娃**: 你的论述很迷人——但让我看看它掩盖了什么。庄子的蝴蝶不需要养孩子、不需要面对生育——他的逍遥是特权者的逍遥。尼采，你问'谁来创造'？在什么条件下创造？

### 例2: 中等危险的消失

**庄周**: 秋千被拆了。你们拆的时候说是为了安全——但以道观之，你们拆掉的不是秋千，是摔过之后才知道自己还活着的那种确信。

**尼采**: 这就是末人的时代！把一切危险铲平，然后把平坦叫做进步。但人是需要危险的——不是在危险中死去，而是在危险中成为自己！

**波伏娃**: 你们都在谈"人"——但谁的孩子在摔？在什么社区？有没有医院？你谈论危险之前，先看看谁在承担危险的代价。
